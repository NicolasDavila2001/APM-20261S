# Modulo 1: Introducción a la automatización de manufactura
El presente módulo aborda el análisis de una arquitectura de automatización industrial estructurada bajo el estándar ISA-95,donde se realizara una evuluacion de la linea de produccion segun la piramide de automatizacion con el fin de poder saber cuales capas ya estan cubiertas y que elementos se encuentran en cada una de estas,se estableceran las etapas del proceso del produccion con el fin de tener un esquema para plantear los diagrmas a presentar en el [modulo 2](https://github.com/NicolasDavila2001/APM-20261S/tree/main/Modulo_2)
